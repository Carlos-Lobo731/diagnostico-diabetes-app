# Sistema de Diagnóstico de Diabetes

App creada con Streamlit que permite evaluar el riesgo de diabetes tipo 2.

## Requisitos

- Python 3.8 o superior
- Streamlit (`pip install -r requirements.txt`)

## Cómo ejecutar

1. Instala los requisitos:
   pip install -r requirements.txt

2. Ejecuta la app:
   streamlit run app.py
